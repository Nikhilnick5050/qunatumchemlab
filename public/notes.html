Featured Tutorial: Your First Calculation

Step 1: Create a New Project

Open QuantumChem and create a new project. Give it a descriptive name related to your calculation.

File > New Project > "My_First_Calculation"

Step 2: Build Your Molecule

Use the molecule builder tool to create your molecular structure. For this example, let's create a water molecule.

Builder > Add Atoms > O Click to place oxygen atom Builder > Add Atoms > H Click to place hydrogen atoms Adjust geometry using optimization tools

Step 3: Choose Calculation Method

Select an appropriate calculation method from the Calculation menu.

Calculation > Method > DFT Functional: B3LYP Basis Set: 6-31G(d)

Step 4: Run Calculation

Start the calculation and monitor progress in the calculation log.

Calculation > Run Calculation Monitor progress in Calculation Log

Step 5: Analyze Results

Once complete, analyze your results using the visualization and analysis tools.

Results > View Molecular Orbitals Results > Calculate Vibrational Frequencies Results > Generate IR Spectrum
